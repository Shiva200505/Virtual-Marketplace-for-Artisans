PRODUCT IMAGES FOR VIRTUAL MARKETPLACE
=========================================

This directory is used to store product images for the Virtual Marketplace application.

Image Naming Convention
----------------------
Images should be named according to the 'image' field value in the database Products table.
For example, if your database contains a product with the image field value "pottery_vase.jpg",
you should save an image file with this exact name in this directory.

Default Images
-------------
If a specific product image is not found, the application will look for a default image
based on the product's category. Default images should be named as follows:

- pottery_default.jpg
- textiles_default.jpg
- woodworking_default.jpg
- jewelry_default.jpg
- paper_crafts_default.jpg
- glass_art_default.jpg

Image Format and Size
--------------------
- Recommended image format: JPG or PNG
- Recommended image size: 300x300 pixels
- Maximum file size: 2MB

If an image is not found, the application will display a placeholder with the product's category.
